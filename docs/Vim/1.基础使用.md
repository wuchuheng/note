## 1 显示行号
### 1.1，临时显示
``` vimscript 
:set number
:set nonumber(撤消显示行号)
```
### 1.2永久显示
``` vimscript 
# vim  /etc/vimrc
set number(加入这行代码
```

## 2 tab键缩进4空格
``` vimscript 
# vim /etc/vimrc
:set ts=4
:set expandtab
:%retab!
```

## 3 vim 打开txt文件乱码
``` vimscript 
# vim /etc/vimcr
set fileencodings=utf-8,gb2312,gbk,gb18030
set termencoding=utf-8
set encoding=prc
```
